Features:
Client-server model
Unified network protocol (see App. B)
Client-side input prediction
Client-side entity interpolation
Server discovery (LAN)
Xor encryption 

Server-side persistent elimination heat-map
Server browser (LAN) 

Instructions:
Use WASD/Direction keys to navigate the menu. Use Enter or Space to confirm. 
Choose connect to go the server list.
Choose a server to connect. 
Wait for the game to start. 
Press esc to pause
Choose disconnect in the pause menu to disconnect. 

The heatmap of the server is stored at "../Heatmaps/EliminationHeatMap.txt".
